Action 1

돼지우동 릴스 바이럴.

인플루언서 협업 + 메타 광고 부스팅으로 도달을 만들고, 6월 강남점 오픈 사전 콘텐츠를 함께 깔아둡니다.

인플루언서 협업
부산 마이크로 3~4명 + 서울 나노 2~3명 (강남 사전 노출) 총 5~7명. 4월 4명 섭외/3명 업로드 패턴 위에서 업로드 가속.
메타 광고 부스팅
4월 광고비 50만원 이월 + 5월 신규 합산으로 본격 집행. 인플루언서 릴스 소재 기반 부스팅, 부산 → 서울·강남 권역 진입.
강남 오픈 사전 콘텐츠
6월 강남점 오픈 런칭일에 맞춰 사전 콘텐츠 작업 미리 진행. 별도 설계 없이 돼지우동 바이럴을 그대로 강남에서도 작동하도록.
측정
"돼지우동" 검색량 280 → 500 · 인스타 릴스 누적 2 → 7건+ · 메타 광고 도달 5만+ (4월 35,044 대비).
Action 2

검색 콘텐츠 누적.

블로그 체험단 40인 이상 업로드 부스팅 + 맛집 키워드 선점 SEO 블로그로 4월에 측정한 콘텐츠 포화도(1.99%, 매우 낮음)를 5월에 본격 끌어올립니다.

블로그 체험단
40인 이상 섭외 → 업로드 부스팅. 4월 10명 섭외 / 2명 업로드 패턴을 5월 업로드 가속으로 정착.
맛집 키워드 선점 SEO 블로그
4월 W5에 대폭락한 광역 맛집 키워드(센텀 맛집·전포 맛집 등)는 NP 메인 키워드로 등록 불가. 블로그 콘텐츠에 녹여 검색 결과 페이지에 미례국밥이 직접 노출되도록 8건 이상 발행.
우선 맛집 키워드
센텀 — 센텀 맛집 · 센텀시티 맛집 · 센텀역맛집 · 재송동맛집
전포 — 전포 맛집 · 전포동맛집 · 부산전포맛집 · 전포역맛집 · 서면전포맛집
측정
월 콘텐츠 발행 71 → 100건+ · 콘텐츠 포화도 1.99% → 3% 이상 · 블로그 체험단 업로드 2 → 40건+ · 맛집 키워드 평균 순위 회복.
Action 3

네이버플레이스 부스팅.

4월에 깔린 NP 셋업(소개문·메인 키워드·예약·쿠폰)을 위클리 추적관찰하며, 광고 집행으로 가속합니다.

NP 최적화
메인 키워드 운영, 소개문·사진·소식 사이클 유지. 4월에 만든 셋업의 효과를 위클리로 추적관찰.
NP 광고 집행
네이버플레이스 광고 본격 집행. 검색 결과 상단 노출 + 매장 클릭 가속.
리뷰 답변 사이클
4월 답변 템플릿 그대로 운영. 센텀 리뷰 능동 회복(73→79, +8.2%) 흐름 5월에도 유지.
측정
전포점 NP 유입 4,410 → 4,600 이상 · 센텀점 NP 유입 1,516 → 1,700 이상 · "전포동 국밥" 1위 사수.
