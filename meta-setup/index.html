왜 비즈니스 매니저(BM)인가

비즈니스 매니저(Meta Business Manager, BM)는 페이스북·인스타·광고 계정·픽셀 등 비즈니스 자산을 한 곳에 모아놓고 관리하는 통합 콘솔이에요. 메타가 최근 UI에서는 "비즈니스 포트폴리오"로 표기하기 시작했지만 같은 의미입니다.

왜 본사가 직접 BM을 만들고 통제권을 가져야 할까요?

  • 자산 소유권 확보 — 광고 대행사가 자기 BM에 본사 페이지·픽셀을 묶어두면, 대행사 변경 시 자산을 못 가져옵니다. 본사 BM에 자산을 모아놓고 대행사를 "파트너"로 초대하는 구조여야 안전.
  • 광고 데이터 영구 보존 — 픽셀이 본사 BM에 있어야 캠페인 종료 후에도 학습 데이터가 남아 다음 캠페인 효율이 누적됩니다.
  • 결제·세금 분리 — 광고비 청구·세금계산서가 본사 명의로 정리됩니다.

이 가이드는 본사가 직접 BM을 신설하는 표준 9단계 절차입니다. 신중히 따라하면 30분, 분기 발동 시 +10~15분.

시작 전 5분 점검

아래 5가지 중 하나라도 빠지면 중간에 막혀 처음부터 다시 해야 합니다. 시작 전 노트북 켜서 한 번씩 확인:

항목확인 방법없으면
A. 페이스북 개인 계정 본인 페이스북 로그인 가능 먼저 페북 가입 (BM은 페북 계정 위에 만들어짐)
B. 페이스북 페이지 어드민 해당 비즈니스 페이지 → 설정 → 페이지 역할 → 본인이 "관리자" 현 관리자에게 권한 양도 요청
C. 사업자등록증 PDF 또는 사진. 사업자등록번호·법인명·주소 확인 등록증 발급 후 진행 (BM 정보 입력에 필요)
D. 법인카드 또는 본사 결제수단 광고비 청구용. 본인 직접 입력 필수 (제3자 공유 X) 광고 운영 보류 또는 제3자 결제 대행 모델 협의
E. 인스타그램 계정 로그인 해당 비즈니스 인스타 앱 로그인 가능 비밀번호 리셋 또는 인스타 운영자 직접 작업

결정적 함정 — 도메인부터 정확히

⚠️ 꼭 business.facebook.com으로 들어가셔야 합니다. 평소 쓰시는 facebook.com(파란 f 로고의 일반 페이스북)이 아니에요. 두 곳은 다른 시스템입니다.

일반 페이스북에 있는 "광고 센터"·"광고 관리자" 메뉴로는 이 가이드의 화면이 안 나옵니다. 그쪽은 페이지 운영자용 단순화 도구, 비즈니스 매니저는 별도 도메인의 본격 콘솔이에요.

처음 진입할 때 가장 많이 막히는 지점이 이 도메인 혼동이에요. 새 탭을 열고 주소창에 business.facebook.com을 직접 입력하거나 여기 클릭해서 새 탭으로 열기.

→ [+ 비즈니스 포트폴리오 만들기]
필드입력값
비즈니스 포트폴리오 이름본사 정식 법인명 또는 브랜드명
이름 / 성본인 이름 (가게 이름이 아니라 운영자 본인)

✅ 검증: 좌상단 비즈니스 셀렉터에 새 BM 표시. 화면에 보이는 BM ID(15자리 숫자)를 메모해두세요.

STEP 3페이스북 페이지 추가

소요 시간: 3~12분 (페이지 신설 케이스 시 +10분)

좌측 사이드바 맨 아래쪽 톱니바퀴 ⚙️
→ "설정" 진입
→ 좌측 메뉴 "계정" 펼치기 ▼ → "페이지"
→ 우상단 파란색 [추가 ▼] → "페이지 추가"
→ 페이지 이름 검색 → 선택 → [페이지 추가]

3가지 분기 케이스

케이스회복
페이지 검색되고 본인이 어드민정상 진행 (3분)
페이지가 다른 BM(예: 외주 대행사)에 종속돼있음현 종속 BM 운영자에 페이지 어드민 양도 요청. 회신까지 1~3일 소요
페이지 자체가 없음페이지 신설부터 (아래 절차, +10분)

페이지 신설 절차 (없을 때만)

facebook.com → 좌측 메뉴 → 페이지 → [새 페이지 만들기]
→ 페이지 이름 / 카테고리 (예: "한식 음식점") / 한 줄 설명
→ [만들기]
→ 프로필·커버 사진 업로드

페이지 신설 직후 BM에서 검색 → 추가.

STEP 4인스타그램 비즈니스 계정 연결

소요 시간: 3~8분 (개인 계정일 때 +5분)

4-1. 인스타가 비즈니스 계정인지 확인

휴대폰 인스타 앱 → 해당 계정 → 프로필 → ☰ → 설정 → 계정 메뉴 확인:

  • "프로페셔널 계정으로 전환" 메뉴 보임 → 개인 계정 (4-2 진행)
  • "프로페셔널 계정" 라벨 보임 → 이미 비즈니스 (4-3 바로 진행)

4-2. 비즈니스 계정 전환 (개인 계정일 때만)

휴대폰 앱에서 직접:

프로페셔널 계정으로 전환
→ 카테고리: 해당 업종 (예: "식당")
→ "비즈니스" 선택 (크리에이터 X)
→ 페이스북 페이지 연결 → 해당 페이지 선택

💡 왜 비즈니스 전환이 필요한가: 메타 광고는 비즈니스 계정만 게재 가능합니다. 개인 계정으론 광고 못 돌려요.

4-3. BM에 인스타 계정 추가

PC로 돌아와서 비즈니스 매니저 설정 화면에서:

설정 → 계정 → Instagram 계정
→ 우상단 [추가 ▼]
→ 인스타 로그인 팝업 → 해당 계정 로그인
→ "페이스북 페이지에 연결" → 페이지 선택 → 연결

STEP 5광고 계정 신설

소요 시간: 4분

설정 → 계정 → 광고 계정
→ 우상단 [추가 ▼] → "새 광고 계정 만들기"
필드값
광고계정 이름"{브랜드명} 메인 광고계정"
시간대Asia/Seoul
통화KRW

[만들기]

생성 직후 사람 할당 화면에서 본인을 "광고주" 권한으로 추가 (캠페인 검토·승인 가능).

💡 광고 계정 신설 직후 "사용 한도" 알림이 뜰 수 있어요. 본사 규모 첫 광고계정이라 정상이에요. 한도는 캠페인 1~2회 운영 후 자동 상승합니다.

STEP 6결제수단 등록

소요 시간: 5분

⚠️ 본인이 직접 입력해야 합니다. 카드 정보는 누구에게도 공유하지 마세요(PCI 준수). 광고 대행사가 카드 정보를 요구하면 거부하세요 — 정상적인 대행사는 본사 BM에 권한만 받고 운영합니다.

신설 광고계정 → 결제 → 결제수단 추가
→ 본사 카드 정보 입력
→ 청구 한도 설정 (월 50만원으로 시작 권장)
→ [저장]

청구 한도 권장: 첫 시작은 월 50만원~100만원이 안전합니다. 캠페인 운영하면서 한도를 늘리는 게 카드사·메타 양쪽 모두에 신뢰가 쌓여요.

STEP 7픽셀(데이터세트) 만들기

소요 시간: 3분

설정 → 데이터 소스 (펼치기 ▼) → 데이터세트
→ 우상단 [추가 ▼] → "픽셀"
→ 픽셀 이름: "{브랜드명} 픽셀"
→ 사이트 URL 입력 → [만들기]

생성 후 표시되는 픽셀 ID(15자리)와 픽셀 코드(JavaScript 박스)를 메모해두세요. 사이트 운영자(또는 사이트 빌더 도구)에 전달해서 사이트 head 영역에 심으면 광고 효과 추적이 시작됩니다.

💡 왜 픽셀이 필요한가: 픽셀은 광고를 보고 사이트에 들어온 사람의 행동(상품 보기, 장바구니, 구매 등)을 추적해서 다음 캠페인 최적화에 사용해요. 픽셀 없으면 광고는 작동하지만 "성과 측정"이 거의 불가능합니다.

자체 사이트가 없는 경우 임시 URL(예: 인스타 프로필 링크 페이지)을 입력하고 일단 만들어만 두세요. 사이트가 생기면 그때 코드를 심으면 됩니다.

STEP 8외부 운영자(파트너) 권한 부여

소요 시간: 3분

광고 대행사·프리랜서 마케터에게 운영을 맡길 때 사용하는 단계예요. 직접 운영한다면 이 Step은 스킵 가능.

설정 → 사용자 (펼치기 ▼) → 파트너
→ 우상단 [추가 ▼] → "파트너 비즈니스에 자산 관리 권한 부여"
→ 파트너 BM ID 입력 (대행사가 알려준 15자리 숫자)
→ [다음]

자산 권한 체크 (4종):

자산권한 등급
광고 계정광고 관리
페이지콘텐츠/광고 관리
Instagram 계정콘텐츠/광고 관리
픽셀광고

[공유]

⚠️ "권한 부여"와 "권한 요청"은 방향이 반대입니다. 헷갈리지 마세요.

· 권한 부여: 우리(본사) → 외부에 권한을 준다 ✓ (대행사 운영 위임 시)
· 권한 요청: 외부 자산을 우리가 받아온다 (반대 방향)

STEP 9검증

소요 시간: 3분

모든 셋업이 정상 작동하는지 확인:

  • BM 좌상단에 신설 BM 표시 (이름 + ID)
  • 설정 → 계정에서 페이지·인스타·광고계정 모두 보임
  • 설정 → 데이터 소스에서 픽셀 보임
  • 광고 계정 → 결제에서 카드·청구 한도 정상
  • (파트너 추가했다면) 사용자 → 파트너에서 외부 BM이 자산 4종 권한 갖고 있음

여기까지 ✓ 다 떨어졌으면 셋업 완료. 첫 캠페인 만들어볼 준비가 됐어요.

막혔을 때 — 자주 나오는 분기 5가지

증상해결
"파트너" 메뉴가 안 보임좌측 "사용자" 항목을 한 번 클릭해서 펼치기 ▼. 그래도 없으면 본인 권한이 BM 관리자(Admin)가 아닌 것 — 현 관리자에게 권한 요청
페이지 검색이 안 됨페이지가 다른 BM에 종속됐거나, 본인이 페이지 어드민이 아닐 가능성. 페이스북 페이지 → 설정 → 페이지 역할에서 본인 권한 확인
인스타 계정 로그인 팝업이 자꾸 닫힘브라우저 팝업 차단 해제 또는 시크릿 창에서 재시도. 두 단계 인증(2FA) 코드 입력 필요할 수 있음
사업자등록번호 거부됨홈택스에서 사업자 상태 확인 → 정상이면 등록증 OCR 오류일 수 있으니 손으로 정확히 입력 → 그래도 거부면 메타 비즈니스 지원 문의
광고계정 신설 시 "한도 초과" 알림본인 페북 계정으로 만들 수 있는 광고계정 수 한도(보통 1개). BM에서 만들면 풀리니, BM 진입했는지 다시 확인

셋업 후 다음 단계

① 픽셀 코드 사이트 심기

Step 7에서 메모한 픽셀 코드를 사이트 <head> 영역에 심어주세요. 카페24·아임웹·식스샵 같은 빌더는 관리자 화면에 "외부 스크립트 삽입" 메뉴가 있어요. WordPress는 플러그인(예: Insert Headers and Footers) 사용.

심은 후 "Meta Pixel Helper" 크롬 확장으로 정상 작동 확인.

② 첫 캠페인은 작게 시작
