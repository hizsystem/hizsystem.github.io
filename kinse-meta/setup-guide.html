메타 비즈니스 계정
0 → 1 세팅 가이드

클라이언트 계정으로 비즈니스 매니저를 만들고, 인스타그램을 연결해, HIZ에 광고 운영 권한을 부여하는 전 과정

예시: 데이나스픽(DANASPICK) 소요 15~20분 2026-07-03 기준
핵심 원칙 — 모든 자산은 클라이언트 소유, HIZ는 "파트너 권한"만 받습니다.
HIZ 계정에 클라이언트 페이지·광고계정을 만들어 넣지 않습니다. 클라이언트가 본인 비즈니스 포트폴리오에 자산을 모아두고, HIZ 비즈니스 ID로 파트너 공유만 걸어줍니다. → 계약이 끝나도 자산은 온전히 클라이언트에게 남습니다.
1

비즈니스 포트폴리오 만들기

business.facebook.com → 비즈니스 설정

  1. 광고 관리자 좌하단 비즈니스 포트폴리오 만들기 클릭
  2. 비즈니스 이름 = 사업자등록증상 정식 상호 / 이름·성 = 담당자 실명 / 업무용 이메일 입력
  3. 생성 후 이메일 인증 링크 클릭
⚠️ 이름이 빨갛게 반려되면 소문자 한 단어(아이디형)는 자주 반려됩니다. 정식 상호(한글) 또는 사업자등록증 상호 그대로 넣으면 대부분 통과합니다. 계속 막히면 뒤에 업종을 붙여 구분(예: "○○ 마케팅").
2

페이스북 페이지 연결

비즈니스 설정 → 계정 → 페이지

  • 기존 페이지 있으면: 추가 → 페이지 소유권 주장
  • 없으면: 추가 → 새 페이지 만들기 (페이지명 = 브랜드명)
  • 마지막 확인 화면에서 약관 동의 체크 → 페이지 만들기
3

인스타그램 연결 (순서 중요)

먼저 인스타 앱에서 전환 → PC에서 연결

① 인스타 앱에서 프로페셔널 전환

  1. 인스타 앱 → 내 프로필 → ≡ → 설정 및 활동
  2. 상단 검색창에 프로페셔널 입력 → 프로페셔널 계정으로 전환
  3. 카테고리 선택 → 계정 유형 비즈니스 선택 → 완료

② PC 비즈니스 설정에서 연결

  1. 계정 → Instagram 계정 → 추가 → 인스타 아이디/비번 로그인
  2. 페이스북 페이지와 연결 → 완료
Tip — "사람 리스트 추가" 토글은 OFF 인스타 로그인에까지 포트폴리오 설정 권한을 줄 필요는 없습니다. 끈 채로 완료.
4

광고 계정 만들기

비즈니스 설정 → 계정 → 광고 계정 → 추가

  • 새 광고 계정 만들기 선택
  • 이름 입력 / 시간대 = (GMT+09:00) 서울 / 통화 = KRW (₩)
  • "이 광고 계정을 사용하는 대상" = 내 비즈니스(포트폴리오) 선택 → 만들기
⚠️ 통화·시간대는 생성 후 변경 불가 반드시 처음에 KRW·서울로 정확히. 그리고 개인 계정에 딸린 기존 광고계정은 "결제 이력이 없으면" 비즈니스로 이동이 안 됩니다 — 이럴 땐 끌어오려 하지 말고 새로 만드는 것이 정답입니다.
5

결제 수단 등록

비즈니스 설정 → 결제 (첫 광고 게시 직전)

  • 결제 수단 추가 → 방금 만든 광고 계정에 연결
  • 클라이언트(브랜드) 명의 카드로 등록 — 광고비는 클라이언트가 직접 부담
지금 안 넣어도 됩니다 파트너 권한(6단계)과 결제는 무관합니다. 다만 첫 광고를 "게시"하는 순간엔 반드시 카드가 등록돼 있어야 합니다. HIZ는 카드 정보를 수령하지 않습니다.
6

HIZ에 파트너 권한 부여 (핵심)

비즈니스 설정 → 사용자 → 파트너 → 추가

  1. 추가 → "비즈니스 ID로 파트너에게 자산 액세스 권한 부여" 선택
  2. HIZ 비즈니스 매니저 ID 입력 → 244896071103516
  3. 넘길 자산 + 권한 배정:
    • 광고 계정 → 광고 관리 (전체 권한)
    • 페이스북 페이지 → 콘텐츠 관리
    • 인스타그램 → 광고 권한
  4. 확인 → HIZ 쪽에 자산이 자동으로 보임
✓ 완료 확인 파트너 목록에 HIZ가 뜨고 "3개 자산 액세스 가능"이면 세팅 완료. 이제 HIZ가 광고를 세팅할 수 있습니다.

⚠️ 자주 막히는 지점 6
