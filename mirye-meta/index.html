미례국밥 · HIZ

미례국밥 메타 광고 초기 세팅 가이드

변미혜 팀장님 · 약 15~20분 소요

한 줄 요약

미례국밥 메타 광고를 처음부터 세팅합니다. 순서는 이렇게:

[비즈니스 매니저 있는지 확인] → [없으면 만들기] → [페이지·인스타 연결] → [픽셀 만들기] → [HIZ 파트너 추가]

댄싱컵과 똑같이 "비즈니스 매니저(BM, Business Manager) 안에 자산 모아놓고, HIZ를 파트너로 초대"하는 구조입니다. 단, 미례국밥은 비즈니스 매니저·자산이 비어있어서 만드는 단계가 추가돼요.

시작 전 준비물

  • 노트북 (PC 권장)
  • 본인 페이스북 계정 로그인
  • 미례국밥 사업자등록증 (비즈니스 매니저 신설 단계에선 안 받아요. 만든 직후 비즈니스 정보 입력 시 필요 — 그때 옆에 두시면 됩니다)
  • 미례국밥 인스타 계정 로그인 정보 (Step 4에서 비즈니스 계정 전환에 사용)
  • HIZ 비즈니스 매니저 ID: 244896071103516 ← Step 6에서 입력

Step 1. 미례국밥 비즈니스 매니저가 이미 있는지 먼저 확인

⚠️ 꼭 business.facebook.com으로 들어가셔야 해요. 평소 쓰시는 facebook.com(파란 f 로고의 일반 페이스북)이 아닙니다. 일반 페이스북의 "광고 센터"·"광고 관리자" 메뉴로는 이 가이드의 화면이 안 나타나요.

  1. 새 탭에서 주소창에 business.facebook.com 직접 입력 (또는 여기 클릭해서 새 탭으로 열기)
  2. 좌상단 비즈니스 셀렉터 클릭
  3. 목록에 "DANCINGCUP_Official" 외에 미례국밥 관련 비즈니스 매니저가 있는지 확인

💡 페이스북 페이지나 인스타 비즈니스 계정을 만든 적 있으면 빈 비즈니스 매니저가 자동 생성돼있는 경우가 종종 있어요. 새로 만들기 전에 꼭 확인.

결과 다음 단계
미례국밥 비즈니스 매니저가 보임 Step 2 스킵, Step 3부터 진행. 그 비즈니스 매니저 선택하고 진행
미례국밥 비즈니스 매니저가 없음 Step 2부터 진행 (새로 만들기)

Step 2. 미례국밥 비즈니스 매니저 신설 (없을 때만)

  1. 좌상단 비즈니스 셀렉터에서 [+ 비즈니스 포트폴리오 만들기] 클릭 (옛 UI에선 "비즈니스 만들기"로 표시)
  2. "비즈니스 포트폴리오 만들기" 창이 뜨면 4개 항목 입력:
    • 비즈니스 포트폴리오 이름: "미례국밥" (또는 본사 정식 법인명)
    • 이름: 팀장님 본인 이름 (예: 미혜)
    • 성: 팀장님 본인 성 (예: 변)
    • 비즈니스 이메일 주소: 팀장님 평소 업무 이메일
  3. [만들기] 클릭

💡 이름·성·이메일은 미례국밥 가게 정보가 아니라 팀장님 본인 연락처예요. 메타가 비즈니스 운영자(=대리인) 등록 시 사용하는 정보라, 본인 이름이 정상입니다.

사업자등록증 정보(사업자번호·주소·전화)는 이 단계에선 안 받아요. 만든 후 비즈니스 정보 메뉴에서 별도 입력하며, HIZ가 광고계정 신설하면서 같이 안내드립니다.

비즈니스 매니저 이름은 나중에 변경 가능합니다. 일단 만들고 진행하세요.

생성 직후 화면(또는 좌상단 비즈니스 셀렉터)에 미례국밥 비즈니스 매니저 ID 숫자가 보일 거예요. 그 숫자를 슬랙으로 HIZ에 알려주세요 (광고계정 셋업 후 wiring에 필요).

Step 3. 페이스북 페이지를 비즈니스 매니저에 연결

  1. 새로 만든 미례국밥 비즈니스 매니저 선택 → 좌측 수직 사이드바 맨 아래쪽 톱니바퀴 ⚙️ 클릭 → "설정" 화면 진입
  2. 좌측 메뉴 계정 펼치기(▼) → 페이지 클릭
  3. 우상단 파란색 [추가 ▼] 드롭다운 → "페이지 추가" 선택
  4. 미례국밥 공식 페이스북 페이지 검색 → 선택
  5. [페이지 추가]

💡 페이지가 검색되지 않으면: 본인 페이스북 계정이 미례국밥 페이지의 관리자가 아닐 수 있어요. 페이지 관리자가 따로 있으면 그분께 비즈니스 매니저 추가 권한 위임 받으셔야 합니다. (페이스북 페이지 → 설정 → 페이지 역할에서 본인을 관리자로 추가)

Step 4. 인스타그램 비즈니스 계정 전환 + 연결

4-1. 인스타 계정을 비즈니스 계정으로 전환 (개인 계정이라면)

💡 왜 하느냐: 메타 광고는 비즈니스 계정만 게재 가능합니다. 개인 계정으론 광고 못 돌려요.

휴대폰 인스타그램 앱에서:

  1. 미례국밥 인스타 계정 로그인
  2. 프로필 → 우상단 ☰ → 설정 및 개인정보
  3. 계정 → 프로페셔널 계정으로 전환
  4. 카테고리: "식당" 선택
  5. 비즈니스 선택 (크리에이터 X)
  6. 페이스북 페이지 연결: 미례국밥 공식 페이지 선택

이미 비즈니스 계정이라면 4-2로.

4-2. 비즈니스 매니저에 인스타 계정 추가

PC에서 설정 화면(좌측 사이드바 ⚙️)으로 돌아와서:

  1. 좌측 메뉴 계정 → Instagram 계정 클릭
  2. 우상단 파란색 [추가 ▼] 드롭다운
  3. 미례국밥 인스타 계정으로 로그인 (팝업 창)
  4. 페이스북 페이지에 연결할지 물으면 연결

Step 5. 픽셀 만들기

💡 왜 하느냐: 픽셀은 우리 광고를 보고 미례국밥 사이트에 들어온 손님이 진짜 들어왔는지, 뭘 보고 갔는지 추적하는 작은 코드입니다. 광고 효과 측정의 기준이에요.

⚠️ 미례국밥에 자체 웹사이트나 예약 페이지가 없으면 픽셀이 작동할 곳이 없어서, 일단 만들어만 놓고 보류해도 됩니다. 사이트 유무는 BC3와 사전에 합의해주세요.

  1. 설정 화면 좌측 메뉴 데이터 소스 펼치기(▼) → 데이터세트 (또는 "픽셀") 클릭
  2. 우상단 파란색 [추가 ▼] 또는 [새 데이터세트 만들기]
  3. 이름: "미례국밥 픽셀"
  4. 웹사이트 URL: 미례국밥 사이트 주소 (없으면 임시로 https://hizsystem.github.io/mirye/ 입력하고 추후 변경)
  5. [만들기]

생성 후 픽셀 ID(15자리 숫자)와 픽셀 코드(JavaScript 코드 박스)가 표시됩니다.

📌 여기서 멈추세요. 코드를 사이트에 직접 심는 건 개발자 작업입니다. 다음 두 가지만 해주세요:

  1. 픽셀 ID 숫자를 슬랙으로 HIZ에 알려주세요
  2. 픽셀 코드 전체(JavaScript 박스)를 복사해서 슬랙에 붙여넣어 주세요

HIZ가 받아서 사이트 운영처에 전달하거나 BC3 개발자가 사이트에 심습니다.

Step 6. HIZ를 파트너로 추가

이 단계는 댄싱컵과 100% 동일합니다.

  1. 설정 화면 좌측 메뉴 사용자 펼치기(▼) → 파트너 클릭
  2. 우상단 파란색 [추가 ▼] 드롭다운 → "파트너 비즈니스에 자산 관리 권한 부여" 선택
  3. HIZ 비즈니스 매니저 ID 입력: 244896071103516
  4. 자산 권한 체크 (있는 자산만, 없으면 스킵):
탭 선택 권한
광고 계정 (이 시점엔 비어있음) 스킵
페이지 미례국밥 공식 페이지 콘텐츠/광고 관리
Instagram 계정 미례국밥 인스타 콘텐츠/광고 관리
픽셀 (데이터세트) 미례국밥 픽셀 광고

💡 광고 계정은 아직 안 만들었어요. 다음에 HIZ가 미례국밥 비즈니스 매니저 안에 만들 예정입니다. 일단 페이지·인스타·픽셀 3종만 공유.

  1. [공유]

끝나면 — HIZ에 한 줄만

슬랙으로:

"미례국밥 세팅 끝났습니다. 비즈니스 매니저 ID: [숫자] 픽셀 ID: [숫자] (픽셀 코드는 별도 메시지로 첨부)"

이 한 줄로 HIZ가 받아서 광고계정 신설부터 첫 캠페인까지 진행합니다.

끝난 후 HIZ가 하는 일 (참고)

팀장님이 Step 6 완료 후, HIZ에서:

  1. 공유 요청 수락
  2. 미례국밥 비즈니스 매니저 안에 광고계정 신설 (HIZ가 셋업, 결제수단은 추후 카드 정보 받아서)
  3. 픽셀 코드를 사이트 운영처/BC3에 전달
  4. 첫 캠페인 기획 → 5월 중 운영 시작

팀장님이 추가로 하실 일: 광고계정 결제수단(본사 카드) 등록은 팀장님 본인이 해야 합니다. 광고계정 신설 후 HIZ가 카드 등록 가이드 별도로 보낼게요.
